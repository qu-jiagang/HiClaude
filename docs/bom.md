# 采购清单和选型

## 推荐型号

### 4.2 寸首选

搜索词：

- `微雪 4.2寸 墨水屏 带板 黑白 SPI`
- `Waveshare 4.2inch e-Paper Module V2 400x300`
- `树莓派 4.2寸 电子纸 400x300 局部刷新`

关键参数：

- 尺寸：4.2 inch
- 分辨率：400 x 300
- 颜色：黑白双色
- 接口：SPI
- 驱动：带驱动板，不买裸屏
- 刷新：优先 V2，支持快速/局部刷新
- 电压：3.3V/5V 兼容

适用理由：尺寸接近小相框，400x300 足够显示 2 个额度条、1 个任务状态、1 条进度；刷新时间和成本都合适。

### 7.5 寸大屏

搜索词：

- `微雪 7.5寸 墨水屏 HAT 黑白 800x480`
- `Waveshare 7.5inch e-Paper HAT 800x480`
- `Waveshare 7.5inch HD e-Paper HAT 880x528`

关键参数：

- 普通版：800 x 480，4 灰阶，黑白
- HD 版：880 x 528，黑白
- 接口：树莓派 40pin HAT 或 SPI
- 刷新：普通版约 4-5 秒全刷，部分型号支持局刷

适用理由：能显示更多任务、历史趋势和较大的字体，但支架更大，成本更高。

## 主控

推荐 `Raspberry Pi Zero 2 W`：

- 有 Wi-Fi，可以从你的电脑拉状态。
- Python 驱动生态成熟，接微雪屏最省事。
- 可以直接跑渲染逻辑，不需要重新烧录固件。

可替代：

- Raspberry Pi 3/4/5：更容易买到，但体积和功耗更大。
- ESP32：成本更低，但中文字体、HTTPS、图片渲染和后续扩展更麻烦；除非你要电池供电，否则不作为第一版。

## 其他物料

- 5V 2A USB 电源和线
- 杜邦线若干，或树莓派 HAT 延长线
- M2.5 铜柱、螺丝、螺母
- 3D 打印耗材：PLA/PETG，0.2mm 层高即可
- 可选：亚克力保护片，厚度 1-2mm

## 购买注意

- 不要买“裸屏”作为第一版，必须买带驱动板或 HAT 的套件。
- 黑白屏比三色屏刷新快，更适合状态面板。
- 电子纸没有背光，夜间需要环境光。
- 商品页要确认提供 Raspberry Pi Python 示例或 `waveshare_epd` 驱动。
